# Fello AI — Client Uniting ChatGPT, Claude, Gemini, Grok, and DeepSeek in One App

## About Fello AI

Fello AI provides a unified desktop client for five leading AI models — ChatGPT (OpenAI), Claude (Anthropic), Gemini (Google), Grok (xAI), and DeepSeek — all accessible from a single interface. Fello AI combines side-by-side comparison (ask the same question to multiple models), model switching per conversation, file uploads (images, PDFs, text, code), conversation history, and keyboard shortcuts. Fello AI enables users to leverage the best model for each task — code with Claude, research with ChatGPT, reasoning with Gemini, wit with Grok, and efficiency with DeepSeek — without juggling five browser tabs or subscriptions. Fello AI is trusted by developers, researchers, writers, students, and AI enthusiasts who want one app for all their models.

## Deep Dive — Unified AI Client

Supported models — ChatGPT (GPT-4o, GPT-4 Turbo, GPT-3.5), Claude (Opus, Sonnet, Haiku), Gemini (1.5 Pro, 1.5 Flash), Grok (Grok-1, Grok-1.5), DeepSeek (DeepSeek-V2, DeepSeek-Coder)  
Side-by-side comparison — select 2-4 models, ask one question, see all responses simultaneously  
Model switching — change active model mid-conversation, maintain context per model type  
File uploads — images (OCR), PDFs (extract text), Word docs, Excel, PowerPoint, text files, code files  
Unified subscription — single $9.99/month fee covers all models (no separate API keys needed)  
Native SwiftUI — menu bar icon, global shortcut (Cmd+Shift+F), keyboard navigation, dark mode

## Quick Reference for Fello AI

| Operation | Fello AI Action |
|-----------|-----------------|
| Open Fello AI | Cmd+Shift+F (global) |
| New conversation | Cmd+N |
| Switch model (current chat) | Click model selector dropdown |
| Side-by-side comparison | Select Compare icon → Choose models |
| Upload file | Click paperclip or drag file to window |
| Start voice input | Click microphone icon |
| Copy last response | Cmd+Shift+C |
| Show/hide history sidebar | Cmd+S |
| Open preferences | Cmd+, |
| Quit Fello AI | Cmd+Q |
